\CWProblem{
В этой лабораторной работе вам предстоит реализовать технику наложения теней. Предстоит работа с рендерингом вне кадра “от лица” направленного источника света в текстуру глубины с использованием расширения Vulkan 1.2 Dynamic Rendering, а также работа с использованием данных о глубине сцены, чтобы создать эффект тени на поверхностях моделей.
}
